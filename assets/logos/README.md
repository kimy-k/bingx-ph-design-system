# BingX logo files

Drop the following SVG files into this directory (rename from the original Chinese filenames):

| Original filename | Rename to |
|---|---|
| `品牌蓝色文字logo.svg` | `bingx-wordmark-blue.svg` |
| `白色文字logo.svg` | `bingx-wordmark-white.svg` |
| `黑色文字logo.svg` | `bingx-wordmark-black.svg` |
| `品牌蓝色icon-logo.svg` | `bingx-icon-blue.svg` |
| `白色icon-logo.svg` | `bingx-icon-white.svg` |
| `黑色icon-logo.svg` | `bingx-icon-black.svg` |

## Usage rules

- Wordmark = full "BingX" logo with text (primary use)
- Icon = symbol only (for favicons, small spaces, watermarks under 24px)
- Blue (`#0058FB`) on light backgrounds
- White on dark backgrounds (default for all assets)
- Black for monochrome print contexts only
- Watermark opacity: 50–60% bottom-right on most assets
